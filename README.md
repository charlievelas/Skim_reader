# Skim_reader
A programme allowing for easy HIPO file skimming using clas12root

Requires ROOT, tested on version 6.30/06.

## How to install
For bash
```bash
git clone https://github.com/charlievelas/TreeUI.git
cd Skim_reader
export SkimR=$PWD
```

## How to run
        root '$TreeUI/loadGUI("/the/location/of/file.root")'
